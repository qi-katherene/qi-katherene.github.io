The inspiration for the design of this chair is that it fulfills the following user requirements:
- Portability
- Comfort

The chair’s collapsible design allows the PVC pipes and connection points to be disassembled into smaller components for easier storage and transportation. The use of lightweight PVC tubing and 3D-printed connectors minimizes the overall weight.
