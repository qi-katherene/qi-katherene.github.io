# Project Requirements
- Fail at the desired force from the pedal (~8.8 ± 1 lbf)
- Break at least 1 cm away from the interface components
- Minimize the mass (<15g)
- Made using only laser-cut ⅛” acrylic, 3D printed PLA, and epoxy

# Final Performance
Our final design was a 2-layer acrylic piece epoxied together. It performed with the following specifications:
- Failure Load = 8lbs
- Broke at distance > 1 cm away from the interface components
- Mass = 9g
- Made of laser cut ⅛” acrylic

Through this project, I learned that PLA's failure load was more unpredictable compared to acrylic through iterative testing. 
The final decision to use only acrylic was based on early testing where acrylic's failure loads were closer to the expected load 
compared to the designs using PLA.
